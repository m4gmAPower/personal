# 家計簿アプリ (Household Budget App)

シンプルで使いやすい、PWA対応の家計簿アプリケーションです。
日々の収入・支出の記録、予算管理、グラフによる分析などが可能です。

## 機能
- **取引管理**: 収入・支出の記録、編集、削除
- **予算管理**: 月ごとの予算設定と消化状況の可視化
- **分析**: 円グラフによる支出内訳の表示
- **データ出力**: CSV形式でのエクスポート
- **PWA対応**: Android等のホーム画面に追加してアプリとして利用可能（オフライン対応）

## 動作環境
- Node.js (v18以上推奨)
- Chrome / Edge などのモダンブラウザ

## セットアップ手順

### 1. インストール
プロジェクトのフォルダで以下のコマンドを実行し、依存パッケージをインストールします。

```bash
npm install
```

### 2. 開発モードで起動
開発サーバーを起動します。

```bash
npm run dev
```

起動後、ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスしてください。

### 3. 本番用ビルドと起動
本番環境向けにビルドして起動する場合の手順です。

```bash
npm run build
npm start
```

## スマートフォン（Android）での使い方 (PWA)
このアプリはPWA（Progressive Web App）に対応しています。

1. ChromeブラウザでアプリのURLを開きます。
2. ブラウザのメニュー（︙）を開きます。
3. 「ホーム画面に追加」または「アプリをインストール」をタップします。
4. ホーム画面に追加されたアイコンから起動すると、アドレスバーのないアプリライクな画面で利用できます。

## ディレクトリ構成
- `src/app`: Next.js App Routerのページコンポーネント
- `src/components`: UIコンポーネント
- `src/hooks`: カスタムフック（データ管理ロジック）
- `public`: 静的ファイル（アイコン、マニフェストなど）
